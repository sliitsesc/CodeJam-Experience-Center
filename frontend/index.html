Lights & Patterns

Trigger a pattern from an external service (curl)
curl -X POST http://localhost:3001/api/lights/1/pattern \
  -H "Content-Type: application/json" \
  -H "X-Source: external-api" \
  -d '{"pattern":"blink"}'
Valid patterns: off · on · blink · fast_blink · heartbeat · strobe · sos
It will appear in the event feed instantly with the external-api badge.
